# Lea_M2 — Predicting HOMO/LUMO energies from structural descriptors

M2 internship archive (cheminformatics). Goal: predict **HOMO / LUMO / GAP** energies
from structural descriptors — π-conjugation length, donor/acceptor groups (D/A),
ECFP fingerprints, Clar sextets and Hückel theory.

Starting datasets: **QMugs**, **PubChemQC**, **COMPAS**.

## 1. Overall project logic

The project always follows the same three-stage flow:

```
   ┌─────────────┐      ┌──────────────────────┐      ┌────────────────────────┐
   │  DATASETS   │  →   │      ANALYSIS        │  →   │     CORRELATIONS       │
   │   (raw      │      │ (descriptor          │      │ (descriptor ↔          │
   │    data)    │      │  extraction)         │      │  HOMO/LUMO/GAP)        │
   └─────────────┘      └──────────────────────┘      └────────────────────────┘
   01_datasets/         02 / 03 / 04 / 05 / 06 / 07    (embedded in the same
                                                         analysis notebooks)
```

1. **Datasets** → the raw files (SMILES + DFT energies), in `01_datasets/`.
2. **Analysis** → each notebook reads a raw dataset and **computes descriptors**
   (π, D/A, ECFP, Clar, Hückel). These computations produce **intermediate working
   files** (enriched with new columns).
3. **Correlations** → these descriptors are related to the energies (regressions, residuals, figures).

> **Raw** files live in `01_datasets/`. The **enriched working** files
> (produced by stage 2) are gathered in `_donnees_intermediaires/`.
